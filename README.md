# 🎲 The Great Programming Journey

A Java-based board game engine designed with strict **Object-Oriented Programming principles**.

This project was developed to explore software architecture, polymorphism, and extensible system design.

---

## 🎮 Game Engine Overview

The engine simulates a turn-based board game where different entities interact with obstacles, tools, and game mechanics.

Key focus:

* Clean architecture
* Extensible design
* Pure polymorphism

---

## 🧠 Core Concepts

The project reinforces several advanced OOP concepts:

* Inheritance
* Polymorphism
* Interface-driven design
* Separation of concerns
* Entity-based architecture

⚠️ The engine intentionally avoids the use of `instanceof`, enforcing **pure polymorphic behavior**.

---

## ⚙️ Features

* Turn-based game system
* Entity interaction mechanics
* Tools and obstacles system
* Persistent game state
* Extensible object hierarchy
